# 高德地图配置说明

## 📍 配置位置

高德地图的 Key 和安全密钥配置在环境变量文件中：

```
ruoyi-ui/
├── .env.development    # 开发环境配置
├── .env.production     # 生产环境配置
└── .env.staging        # 预发布环境配置
```

## 🔧 配置文件说明

### src/config/amap.js

这个文件从环境变量中读取配置：

```javascript
export default {
  // 从环境变量读取 Key
  key: process.env.VUE_APP_GAO_DE_KEY,
  
  // 从环境变量读取安全密钥
  securityJsCode: process.env.VUE_APP_GAO_DE_SECURITYJSCODE,
  
  // 其他配置...
}
```

## 📝 修改配置步骤

### 1. 修改环境变量

编辑对应的 `.env` 文件：

```bash
# 开发环境
vim ruoyi-ui/.env.development

# 生产环境
vim ruoyi-ui/.env.production
```

### 2. 重启开发服务器

```bash
# 停止当前服务（Ctrl + C）
# 重新启动
cd ruoyi-ui
npm run dev
```

### 3. 验证配置

打开浏览器控制台，检查是否有地图加载错误。

## 🌐 高德地图控制台

- 控制台地址：https://console.amap.com/dev/key/app
- 在控制台可以：
  - 查看 Key 使用情况
  - 配置域名白名单
  - 获取安全密钥
  - 查看调用统计

## ⚠️ 注意事项

1. **环境隔离**：开发环境和生产环境可以使用不同的 Key
2. **安全性**：不要将 Key 提交到公开的代码仓库
3. **域名白名单**：在高德控制台配置允许的域名
4. **重启生效**：修改 `.env` 文件后必须重启服务才能生效
5. **配额限制**：注意 API 调用次数限制

## 🔍 调试方法

### 检查环境变量是否加载

在组件中添加调试代码：

```javascript
console.log('高德地图Key:', process.env.VUE_APP_GAO_DE_KEY);
console.log('安全密钥:', process.env.VUE_APP_GAO_DE_SECURITYJSCODE);
```

### 检查地图加载状态

打开浏览器控制台，查看是否有以下错误：

- `INVALID_USER_KEY`：Key 无效
- `DAILY_QUERY_OVER_LIMIT`：超过每日调用限制
- `INVALID_USER_SCODE`：安全密钥错误
